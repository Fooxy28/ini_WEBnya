# Web Task and Schedule Management

Aplikasi web sederhana untuk mengelola tugas dan jadwal harian Anda. Proyek ini dibuat untuk memenuhi tugas mata kuliah Evaluasi Desain Perangkat Lunak.

## ğŸ“ Deskripsi

Web Task and Schedule Management adalah aplikasi berbasis web yang membantu pengguna untuk mencatat, memantau, dan mengelola tugas serta jadwal mereka. Dengan antarmuka yang bersih dan mudah digunakan, pengguna dapat dengan mudah melihat progress harian mereka dan mengatur prioritas tugas.

## âœ¨ Fitur Utama

- **Autentikasi Pengguna**: Halaman Login dan Registrasi untuk keamanan akses.
- **Dashboard Interaktif**: Tampilan ringkasan progress tugas harian dan tugas yang sedang berjalan (*In Progress*).
- **Manajemen Tugas (Task)**:
  - Menambahkan tugas baru.
  - Melihat detail tugas.
  - Mengedit informasi tugas.
  - Menampilkan daftar tugas.
- **Manajemen Jadwal (Schedule)**:
  - Menambahkan jadwal kegiatan.
  - Melihat detail jadwal.
  - Mengedit jadwal.
- **Profil Pengguna**: Halaman untuk melihat dan mengedit profil pengguna.
- **Desain Responsif**: Tampilan yang menyesuaikan dengan berbagai ukuran layar.
